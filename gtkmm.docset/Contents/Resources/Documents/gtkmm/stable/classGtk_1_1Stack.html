Detailed Description

A container that shows only one child at a time.

Gtk::Stack does not provide a means for users to change the visible child. Instead, the Gtk::StackSwitcher or Gtk::StackSidebar widget can be used with Gtk::Stack to provide this functionality.

If it is homogeneous, the Gtk::Stack will request the same size for all its children. If it isn't, the stack may change size when a different child becomes visible.
